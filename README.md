# Flux-Checkbox-Example

Example app for Flux pattern.

- [christianalfoni WebApp Enthusiast](http://christianalfoni.github.io/javascript/2014/08/20/react-js-and-flux.html "christianalfoni WebApp Enthusiast")

## Installation

```
npm install
# ractive
npm run build
npm run watch
open index.html
# vue.js
npm run vue-build
npm run vue-watch
open vue/indexhtml
```
